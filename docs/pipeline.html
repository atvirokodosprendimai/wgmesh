wgmesh

Decentralized WireGuard mesh networking.
Zero config. End-to-end encrypted.

⭐ GitHub Quick Start → Pipeline Dashboard

Built for production mesh networking

🌐

DHT Discovery

Peers find each other through a distributed hash table — no central coordinator needed.

🔗

NAT Traversal

Automatic hole-punching and relay fallback connects peers behind firewalls and NATs.

⚡

Zero Config

Single binary. No orchestration server. Start a mesh in seconds with one command.

🔒

End-to-End Encrypted

All traffic flows through WireGuard tunnels. No plaintext ever leaves a node.

Install in seconds

Docker
docker run -d --name wgmesh \ --cap-add=NET_ADMIN --network=host \ ghcr.io/atvirokodosprendimai/wgmesh:latest
Go Install
go install github.com/atvirokodosprendimai/wgmesh/cmd/wgmesh@latest
From Source
git clone https://github.com/atvirokodosprendimai/wgmesh cd wgmesh && go build -o wgmesh ./cmd/wgmesh
